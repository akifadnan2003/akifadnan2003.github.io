Overview

Turkish is a highly agglutinative language — a single word form can carry many morphological tags simultaneously, making part-of-speech disambiguation significantly harder than in English. This project trains a Conditional Random Fields (CRF) sequence model on the UD Turkish-IMST Treebank to predict Universal POS tags for each token in a sentence.

The model achieves 90.93% accuracy and 90.80% weighted F1 on the official test split. A Flask web application allows live inference: input any Turkish sentence and receive per-token POS predictions alongside CoNLL-U output.

Features fed to the CRF include surface form, character n-grams, suffix patterns, surrounding token context, and morphological feature strings from the treebank. The model learns to resolve ambiguity at the sequence level, using context from neighbouring tokens rather than deciding each word in isolation.

Punctuation, conjunctions, and auxiliaries are disambiguated with near-perfect accuracy. PROPN and NUM are the hardest classes — proper nouns share surface forms with common nouns and numbers can appear as adjectives or determiners in Turkish.

Dataset

The UD Turkish-IMST Treebank is part of the Universal Dependencies project. It provides morphologically rich annotations in CoNLL-U format with UPOS tags, morphological features, lemmas, and dependency relations for Turkish sentences drawn from news and literary text.
