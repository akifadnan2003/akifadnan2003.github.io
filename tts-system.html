IVR Voice Generation System

Back to Projects

Overview

A Python application that programmatically generates all IVR (Interactive Voice Response) greeting audio files for a multi-country call centre operation. Rather than recording greetings manually with voice actors for each market, the system uses the Google Cloud Text-to-Speech API to produce natural-sounding, locale-specific audio from text templates.

The generator covers 20+ country domains, each requiring gender-specific (male/female) voice variants and locale-accurate pronunciation. Output files are produced in both MP3 and WAV formats to satisfy the format requirements of different telephony systems.

Features

Locale-Accurate Pronunciation

Each domain maps to an exact BCP-47 language tag (e.g. fr-BE for French Belgium) ensuring the Cloud TTS engine uses the correct regional accent and phoneme rules rather than a generic fallback.

Dual-Gender Voices

Every country domain produces both a male and female greeting, giving telephony administrators the option to select the preferred voice for each market without re-running the generator.

Dual-Format Output

Audio is exported as MP3 for general use and as WAV (PCM) for telephony systems that require uncompressed audio. Both formats are generated in a single API call cycle.

Batch Generation

A single script run generates all domain-language-gender combinations in sequence, producing the full library of greetings without any manual steps between domains.
