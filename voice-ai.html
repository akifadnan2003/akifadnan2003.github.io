Overview

A full-stack voice AI platform that handles every stage of customer call intelligence: real-time voice interaction via phone, automatic speech transcription, knowledge-base-grounded Q&A, and post-call analytics. The system is designed to serve as an intelligent call agent that can answer product and order questions without any human involvement.

The platform integrates Twilio for telephony, Deepgram for low-latency speech-to-text, and a custom RAG engine backed by order and product vector stores. All call data is persisted in Supabase with PII automatically masked before storage, ensuring GDPR-compliant data handling.

Features

Real-Time Voice Agent

Twilio handles incoming phone calls; Deepgram streams speech-to-text in real time. The AI engine processes the transcription and generates a spoken reply — all within sub-second latency.

RAG-Powered Q&A

Order and product knowledge bases are stored as dense vector embeddings. Customer questions are embedded and matched semantically, ensuring factually grounded answers rather than hallucinated responses.

PII Masking Pipeline

A multi-stage PII processor detects and masks names, phone numbers, addresses, and order IDs before any data is written to Supabase. A master PII filter provides a final safety net.

Call Analytics

Post-call analytics module classifies call intent, extracts key entities, and writes structured summaries to the database for quality monitoring and trend analysis.

Helpdesk Ticket Sync

Unresolved calls can automatically generate helpdesk support tickets with a full transcript and extracted context attached, ensuring seamless handover to human agents.

Web Chat Fallback

A Streamlit-based web chat UI exposes the same AI engine through a browser interface — useful for testing, internal use, or customers who prefer text over voice.
